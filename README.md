# dl-typescript

A light implementation of TypeScript, focused on simplicity and performance.

## Features
- Type inference
- Basic type checking
- Compiler options for optimization

## Installation
To install this project, run:

```
npm install dl-typescript
```

## Usage
To use this TypeScript implementation, simply import it in your project:

```javascript
import { compile } from 'dl-typescript';
```
